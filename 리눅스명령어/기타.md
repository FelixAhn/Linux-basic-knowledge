## date
```bash
현재의 날짜와 시간을 출력

- date : 현재 날짜와 시간 출력
    - 예: date
- date -d "[날짜 또는 시간]" : 지정한 날짜/시간으로 출력
    - 예: date -d "yesterday" (어제 날짜 출력)
    - 예: date -d "next Monday" (다음 월요일 날짜 출력)
- date '+%Y-%m-%d %H:%M:%S' : 지정한 형식으로 날짜/시간 출력
    - 예: date '+%Y-%m-%d %H:%M:%S' (2025-01-07 14:45:00과 같은 형식 출력)
- date -s "[날짜 또는 시간]" : 시스템 시간을 변경
    - 예: date -s "2025-01-07 14:45:00"
```

## seq(Sequence)
```bash
지정한 규칙으로 숫자열을 출력하는 도구

- seq [종료값] : 1부터 종료값까지 숫자 출력
    - 예: seq 10 (1부터 10까지 출력)
- seq [시작값] [종료값] : 시작값부터 종료값까지 출력
    - 예: seq 5 10 (5부터 10까지 출력)
- seq [시작값] [증가값] [종료값] : 증가값 단위로 출력
    - 예: seq 1 2 10 (1, 3, 5, 7, 9 출력)
- seq -w : 출력 숫자를 같은 폭으로 정렬
    - 예: seq -w 1 10 (01, 02, ..., 10 출력)
- seq -f "[포맷]" : 출력 형식을 지정
    - 예: seq -f "%.2f" 1 0.5 2 (1.00, 1.50, 2.00 출력)
```

## more
```bash
한 화면씩 지정한 파일의 내용을 출력하는 도구

- more [파일명] : 지정한 파일 내용 출력
    - 예: more myfile.txt
- 스크롤 명령:
    - Enter : 한 줄씩 스크롤
    - Space : 한 화면씩 스크롤
    - q : 종료
    - /[검색어] : 지정한 검색어 찾기
```

## watch
```bash
지정한 명령어를 지정한 시간(초)마다 재실행하여 화면에 출력하는 도구

- watch [명령어] : 기본적으로 2초 간격으로 명령 실행
    - 예: watch date (2초마다 현재 시간 출력)
- watch -n [초] [명령어] : 지정한 간격(초)으로 명령 실행
    - 예: watch -n 5 ls (5초마다 디렉토리 목록 출력)
- watch -d [명령어] : 출력 결과의 변경된 부분 강조
    - 예: watch -d free (메모리 상태 변화 강조)

* watch + uptime (간단한 모니터링)
```

## crontab
```bash
리눅스의 잡 스케줄러의 내용을 출력하거나 편집할 수 있는 도구

- crontab -l : 현재 사용자의 crontab 목록 출력
- crontab -e : 현재 사용자의 crontab을 편집
- crontab -r : 현재 사용자의 crontab 삭제 (매우 주의: 복구 불가)
```
