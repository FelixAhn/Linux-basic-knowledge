## find
```bash
지정한 파일명 또는 정규표현식을 이용하여 파일을 검색

- find [경로] -name [파일명] : 지정한 파일명을 가진 파일 검색
- find [경로] -iname [파일명] : 대소문자를 구분하지 않고 파일 검색
- find [경로] -type [파일 유형] : 특정 유형의 파일 검색 (f: 일반 파일, d: 디렉토리)
- find [경로] -size [+/-크기] : 지정한 크기보다 크거나 작은 파일 검색 (예: -size +1M)
- find [경로] -exec [명령어] {} \; : 검색 결과에 대해 명령어 실행
    - 예: find /tmp -name "*.log" -exec rm {} \; (로그 파일 삭제)
- find [경로] -mtime [+/-일수] : 지정한 일수 전후로 수정된 파일 검색
```

## which
```bash
환경변수 PATH에 등록된 디렉토리에 있는 명령어를 찾아주는 도구

- which [명령어명] : 명령어의 전체 경로 출력
    - 예: which python (Python 실행 파일 위치 출력)
```

## grep (Global / Regular Expression / Print)
```bash
텍스트 검색 기능을 가진 도구
파일이나 표준 입력을 검색하여 지정한 정규 표현식과 맞는 줄을 출력

- grep [패턴] [파일명] : 파일에서 패턴에 맞는 줄 출력
- grep -i [패턴] [파일명] : 대소문자를 구분하지 않고 검색
- grep -r [패턴] [경로] : 지정한 경로 내 파일들을 재귀적으로 검색
- grep -c [패턴] [파일명] : 패턴과 일치하는 줄의 개수 출력
- grep -v [패턴] [파일명] : 패턴과 일치하지 않는 줄 출력
- grep -n [패턴] [파일명] : 검색 결과에 줄 번호 출력
- grep -l [패턴] [경로] : 패턴이 포함된 파일명만 출력
```

## history
```bash
명령어를 수행한 목록을 출력/조작

- history : 명령어 기록 출력
- history [숫자] : 최근 [숫자]개의 명령어 출력
- ![번호] : 지정한 번호의 명령어 실행
    - 예: !42 (42번 명령어 실행)
- !! : 직전에 실행한 명령어 재실행
- ![문자열] : 지정한 문자열로 시작하는 가장 최근 명령어 실행
    - 예: !grep (최근에 실행한 grep 명령어 실행)
- history -c : 명령어 기록 삭제
```
