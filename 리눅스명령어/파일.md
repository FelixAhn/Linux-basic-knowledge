## touch
```bash
지정한 이름의 비어있는 파일을 생성하거나 파일이 있는 경우 타임스태프를 업데이트

- touch [파일명] : 파일 생성 또는 타임스탬프 업데이트
- touch -c [파일명] : 파일이 없으면 새로 생성하지 않고, 기존 파일의 타임스탬프만 업데이트
- touch -t [시간 형식] [파일명] : 지정한 시간으로 타임스탬프를 업데이트 (예: YYYYMMDDhhmm.ss)
```

## cat (Catenate)
```bash
지정한 파일의 내용을 출력

- cat [파일명] : 파일 내용을 출력
- cat [파일명 1] [파일명 2] > [결합 파일명] : 두 파일을 결합하여 새로운 파일 생성
- cat -n [파일명] : 각 줄에 번호를 추가하여 출력
```

## head
```bash
지정한 파일의 1라인부터 지정한 라인까지 출력 (기본 지정값 10)

- head [파일명] : 파일의 처음 10줄 출력
- head -n [지정한 라인 수] [파일명] : 지정한 줄 수만큼 출력
```

## tail
```bash
지정한 파일의 마지막 라인부터 지정한 수 만큼의 라인을 출력 (기본 지정값 10)

- tail [파일명] : 파일의 마지막 10줄 출력
- tail -n [지정한 라인 수] [파일명] : 지정한 줄 수만큼 마지막 라인 출력
- tail -f [파일명] : 파일의 실시간 업데이트 내용을 출력 (* 로그 모니터링 등에 사용)
- tail -F [파일명] : 파일이 삭제되거나 교체된 경우에도 계속 실시간 내용을 출력
```

## cp (Copy)
```bash
지정한 파일을 지정한 위치와 이름을오 복사

cp [원본 파일] [대상 경로/파일명] : 파일을 복사
cp [원본 파일 1] [원본 파일 2] ... [대상 디렉토리] : 여러 파일을 대상 디렉토리에 복사
cp -r [원본 디렉토리] [대상 디렉토리] : 디렉토리를 복사하며, 하위 디렉토리와 파일 포함
cp -f [원본 파일] [대상 파일] : 대상 파일이 이미 존재하면 강제로 덮어쓰기
cp -p [원본 파일] [대상 파일] : 원본 파일의 속성(소유권, 권한, 타임스탬프 등)을 유지하며 복사
cp -rfp [원본 파일/디렉토리] [대상 경로] : 
    - `-r` : 디렉토리를 포함한 모든 하위 파일과 디렉토리를 복사
    - `-f` : 복사 과정에서 파일을 강제로 덮어쓰기
    - `-p` : 파일의 원래 속성을 유지하며 복사
```

## mv (Move)
```bash
지정한 파일을 지정한 위치와 이름으로 이동

- mv [원본 파일] [대상 파일] : 파일을 이동하거나 이름 변경
- mv [파일 1] [파일 2] ... [대상 디렉토리] : 여러 파일을 대상 디렉토리로 이동
- mv -i [원본 파일] [대상 파일] : 대상 파일이 존재할 경우 덮어쓰기 전에 확인 요청
- mv -f [원본 파일] [대상 파일] : 강제로 덮어쓰기
```

## rename
```bash
지정한 규칙에 따라 여러 개의 파일 이름을 변경

- rename [변경 전 문자열] [변경 후 문자열] [파일 패턴] : 파일명에서 문자열 변경 (예: rename 'old' 'new' *.txt)
- rename -n [변경 전 문자열] [변경 후 문자열] [파일 패턴] : 실제 변경 없이 어떤 파일이 변경될지 출력
- rename -v [변경 전 문자열] [변경 후 문자열] [파일 패턴] : 변경 과정을 자세히 출력
```

## rm (Remove)
```bash
지정한 파일을 삭제

- rm [파일명] : 파일 삭제
- rm -r [디렉토리명] : 디렉토리와 하위 파일/디렉토리 삭제
- rm -f [파일명] : 파일을 강제로 삭제 (확인 요청 없음)
- rm -rf [디렉토리명] : 디렉토리와 모든 하위 내용을 강제로 삭제
```

## less
```bash
상하로 커서 이동이 가능한 파일보기

- less [파일명] : 파일 내용을 스크롤 가능하게 보기
- less +[라인 번호] [파일명] : 지정한 라인부터 파일 보기 시작
- [키] : 
    - `↑` / `↓` : 한 줄씩 위아래 이동
    - `Space` : 한 화면 아래로 이동
    - `b` : 한 화면 위로 이동
    - `/[검색어]` : 파일에서 특정 텍스트 검색
    - `q` : 종료
```

## ln(Link)
```bash
지정한 파일에 대한 심보릭링크나 하드링크 생성

- ln [원본 파일] [링크 파일] : 하드 링크 생성
- ln -s [원본 파일] [링크 파일] : 심볼릭 링크 생성
- ln -f [원본 파일] [링크 파일] : 이미 존재하는 링크 파일을 강제로 덮어쓰기
```

## paste
```bash
지정한 파일들의 행을 읽어 탭으로 구분하여 병합

- paste [파일명 1] [파일명 2] : 두 파일의 각 행을 탭으로 구분하여 병합
- paste -s [파일명] : 여러 행을 한 행으로 병합
- paste -d [구분자] [파일명 1] [파일명 2] : 특정 구분자로 파일의 행 병합 (예: `-d ','`는 콤마로 병합)
```

## dd (Dataset Dafinition)
```bash
블록 단위로 데이터셋을 정의하여 파일을 쓰고 읽음

- dd if=[입력 파일] of=[출력 파일] : 입력 파일을 읽어 출력 파일에 쓰기
- dd if=[입력 파일] of=[출력 파일] bs=[블록 크기] : 지정한 블록 크기로 데이터 읽고 쓰기
- dd if=/dev/zero of=[파일명] bs=1M count=100 : 크기 100MB의 빈 파일 생성
- dd if=[입력 파일] of=[출력 파일] status=progress : 진행 상태를 출력하며 복사
```

## tar (Tape ARchive)
```bash
지정한 데이터 및 디렉토리를 하나의 파일로 만듬

- tar -cvf [아카이브 파일명.tar] [대상 파일/디렉토리] : 파일이나 디렉토리를 tar로 묶음
- tar -xvf [아카이브 파일명.tar] : tar로 묶인 파일을 해제
- tar -czvf [아카이브 파일명.tar.gz] [대상 파일/디렉토리] : gzip을 사용해 tar 파일 압축
- tar -xzvf [아카이브 파일명.tar.gz] : gzip으로 압축된 tar 파일 해제
- tar -tvf [아카이브 파일명.tar] : tar 파일 내부 내용을 목록으로 출력
```
