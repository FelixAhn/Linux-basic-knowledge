## 대화식 쉘 스크립트
```bash
쉘 스크립트 실행 후 사용자에게 입력을 받은 숫자/문자를 처리
ex) 정해진 숫자 맞추기 게임
```

### 예시1
```bash
vi read_test.sh

#!/bin/bash

# read -sp "화면에표시할문자" 변수이름
# p와 sp의 차이점은 secret 모드 (입력창에 표시되지 않음)
read -p "문자입력 : " ANY
read -sp "문자입력 : " ANY2

echo ${ANY}
echo ${ANY2}
```

### 예시2 (숫자 맞추기 게임)
```bash
vi game.sh

#!/bin/bash

GOAL=$(($RANDOM% 100 + 1))
CNT=0

while true
do
    read -p "1과 100사이의 숫자를 입력 : " NUM
    CNT=$(($CNT + 1))
    if [ ${NUM} -gt ${GOAL} ]
    then
        echo "입력한 숫자가 더 큼"
    elif [ ${NUM} -lt ${GOAL} ]
    then
        echo "입력한 숫자가 더 작음"
    elif [ ${NUM} -eq ${GOAL} ]
    then
        ech0 "숫자 맞춤!"
        exit 0
    fi
done
echo "입력횟수 : ${CNT}"    
```
